# Text File Analyzer

## Overview

Text File Analyzer is a Python program that reads a text file and performs basic text analysis. The program calculates the number of characters, lines, and words in the file. It also determines the frequency of each word and displays the five most common words.

## Features

* Reads text from a file.
* Counts the total number of characters.
* Counts the total number of lines.
* Counts the total number of words.
* Calculates the frequency of each word.
* Displays the top 5 most common words.
* Removes punctuation and ignores letter case for accurate analysis.

## Requirements

* Python 3.13

## How to Run

1. Place the text file (`PythonSavvy.txt`) in the same folder as `main.py`.
2. Open a terminal in the project directory.
3. Run the program:

```bash
python main.py
```

* File Handling
* String Manipulation
* List Comprehensions
