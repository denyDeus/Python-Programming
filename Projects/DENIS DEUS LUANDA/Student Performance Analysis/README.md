# Student Performance Analysis System

## Overview

The Student Performance Analysis System is a Python-based data analysis program that reads student performance data from a CSV file, calculates key statistical measures, and visualizes the results using charts.

The program uses the Pandas library for data processing and Matplotlib for data visualization.

---

## Features

- Load student performance data from a CSV file.
- Calculate the average score for each student.
- Compute statistical measures:
  - Mean Score
  - Median Score
  - Highest Score
  - Lowest Score
- Generate a Bar Chart of student average scores.
- Generate a Pie Chart showing score distribution.
- Generate a Histogram showing score frequency distribution.
- Display results in a readable format.

---

## Technologies Used

- Python 3
- Pandas
- Matplotlib
